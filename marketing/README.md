# MoonCut Marketing Materials

This directory contains public-facing MoonCut materials. Keep it separate from application runtime assets so campaign files are easy to find and do not affect product builds.

- `assets/brand/`: approved reusable brand assets, including the transparent MoonCut logo.
- `pages/`: source files, exports, and supporting assets for promotional pages.
- `presentations/`: presentation source files and exported PDFs.

Name deliverables as `YYYY-MM-DD-topic-version.ext`, for example `2026-07-11-product-intro-v1.pdf`.
