# MoonCut AI Agent Deck V4 Brief

- 目标：修剪 v3 的标题与 storyline，让每一页都能顺着黑客松答辩逻辑讲清楚。
- 禁用句式：标题与 key point 避免“不是……而是……”。
- 叙事链：具体用户场景痛点 -> 创意方案 -> 48 小时系统实现 -> 产品如何工作 -> 场景验证 -> 竞品类比 -> 商业增长和护城河 -> 团队与投资人诉求。
- 风格：沿用 v3 的明亮科技艺术风，每页 3-4 个板块，文字更明显，画面更真实，减少抽象摆设。
- 商业分析：用 CapCut/剪映、Descript、Runway、HeyGen 等市场类别做轻量竞品定位，说明 MoonCut 的口播表达 Agent 切入点、盈利路径和护城河。
- 输出：10 页 full-bleed PDF/PPTX。
