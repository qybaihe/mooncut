# MoonCut Web

Vue 3 + TypeScript + Vite 构建的口播创作前端原型。当前不依赖后端，包含完整的本地演示状态。

## 本地运行

```bash
npm install
npm run dev
```

生产构建：

```bash
npm run build
```

## 已实现流程

- 剪辑台：视频拖拽/选择、真实本地预览、剪辑选项、模拟处理进度、成片结果与对比。
- 录制间：助手对话、建议选择、口播稿生成与编辑、本地自动保存、提词录制、录制结果一键送入剪辑台。
- 浏览器有权限时使用摄像头和 `MediaRecorder` 本地录制；无权限时自动进入可完整体验的演示录制模式。
- 桌面端顶部导航与移动端底部导航，两套布局均已适配。
- 创作搭子“小月”：根据等待、构思、处理、录制和完成状态切换 HappyDog 帧动画；轻触会跳跃、冒爱心并提升本地保存的开心值。

宠物素材来源与生产使用提示见 `public/pet/SOURCE.md`。
