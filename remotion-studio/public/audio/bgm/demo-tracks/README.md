# MoonCut Hackathon BGM Demo Pack

10 首纯音乐已经统一为 48 kHz、双声道、192 kbps MP3，并归一化到约 -16 LUFS。目录条目、推荐混音增益、BPM、哈希和来源位于 `../../../../src/data/bgm-library.json`。

- `see-line-funk.mp3` — See Line Funk / Alejandro Magaña / 99 BPM
- `funky-fit.mp3` — Funky Fit / Ahjay Stelino / 99 BPM

## 中文口播参考锚点

- `nothing-to-fear.mp3` — Nothing To Fear / Dexter Britain / 122 BPM

这首在中文知识口播歌单中多次出现。原作者将其列入 Creative Commons 曲目，Demo 按 [CC BY-NC-SA 4.0](https://creativecommons.org/licenses/by-nc-sa/4.0/) 使用；署名为 `Nothing To Fear — Dexter Britain (dexterbritain.com)`。[作者页面](https://dexterbritain.com/creative-commons) / [曲目页面](https://dexterbritain.bandcamp.com/track/nothing-to-fear)

## Remotion 快速切换

- `TalkingHeadDemo`：科技/信息流
- `TalkingHeadDemoGroove`：轻松网感
- `TalkingHeadDemoEmotional`：情绪叙事

也可以给 `TalkingHeadDemo` 传入任意目录中的 `bgmTrackId`，并用 `bgmGainOffsetDb` 临时微调整体音乐响度。
