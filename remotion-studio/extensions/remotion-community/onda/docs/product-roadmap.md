# Product roadmap — v1 scope

## Primitives (~30)

Fade in/out, slide (4 directions), scale-in, blur-reveal, typewriter, word-by-word text, count-up number, draw-on (SVG path), wipe, mask-reveal, stagger-group wrapper, spring-pop (restrained), rotate-in, gradient-shift background, grain overlay, etc.

## Scene blocks (6–10) — the launch "wow"

Title card, lower-third, stat/number reveal, animated list build, logo sting, quote card, section transition, end card. **Blocks compose primitives** — they must never reinvent motion the primitives already handle.

## Sequencing

Prioritize the scene blocks for visual impact, but build them *from* the primitives so the motion language stays coherent. Everything is MIT.

## Per-initiative work

Active and historical execution plans live under [techspecs/](techspecs/). Each is a folder with a `design.md` (the decisions and shape of the change) and a `roadmap.md` (execution plan / milestones).
