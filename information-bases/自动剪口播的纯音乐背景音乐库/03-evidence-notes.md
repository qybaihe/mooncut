# Evidence Notes

## Problem and Context

- `Confirmed`：现有 Remotion 作品的声音全部来自口播视频，没有独立 BGM、循环、淡入淡出或人声闪避路径。
- `Inferred`：自动剪辑产品需要的不只是“免费可用”，还必须区分“允许同步到成片”与“允许把原始 MP3 内置/再分发”。后者才适合离线 Core Pack。
- `Confirmed`：CC0 官方许可覆盖复制、修改、分发与商用，无需署名，是当前需求最直接的授权工具。

## Existing Solutions

- `Confirmed`：HoliznaCC0 的 FMA 单曲页同时提供音频下载、逐曲 CC0 声明和纯音乐标记。
- `Confirmed`：Incompetech 的 CC BY 4.0 目录也允许再分发，是扩充曲库的可靠备选，但必须自动生成署名。
- `Confirmed`：Mixkit、Pixabay、Scott Buckley 和 YouTube Audio Library 更适合“把音乐同步进最终作品”，不适合作为原始 MP3 内置库。

## Recent Changes

- `Confirmed`：Mixkit User Terms 在 2025-10-02 更新，明确禁止聚合素材库存和构建相似服务。
- `Confirmed`：Pixabay 当前 FAQ 详细说明了音乐 Content ID 盾牌、申诉和证书流程。
- `Confirmed`：FreePD.com 已于 2025 年关闭，无法再把官网作为稳定的新下载源。

## China Platform Pattern Update — 2026-07-10

- `Observed`：抖音当前知识干货教程将合适 BGM 概括为“无歌词、节奏平稳、轻快不沉闷、全程衬托人声”；另一口播创作者认为硬核知识可不加音乐，情绪/认知/人生感悟需要音乐作为情绪主线。
- `Observed`：当前高收藏口播歌单的两个核心簇是：①中文情绪钢琴/间奏，如《幻昼》《晴（间奏版）》《繁华的寂静》《喜剧之王背景音乐》；②复古轻松 groove，如 `Sally`、`Paloma`、`Old School`、`Funk Caravan`、`Get The Groove`。
- `Observed`：知识口播样本还使用 `Time Back`、`Nothing To Fear`、`A Little Story`、`Monsieur Melody`、`Catch That Groove` 等海外商业/production music。因而“中国接受度”更接近平台熟悉感与制作质感，而不是作曲者国籍。
- `Observed`：小红书高收藏样本同样把 BGM 按情绪、Vlog、口播、旅行或网感类型整理；商业 IP/观点类偏克制、深色、电影感和渐进，而不是轻松 ukulele 或普通 lo-fi。
- `Observed`：走心内容偏 60–90 BPM 的钢琴、弦乐与熟悉旋律；日常/知识偏 90–115 BPM 的轻 funk、lounge、clean groove；信息流偏 105–125 BPM 稳定鼓点；120 BPM 以上的强能量更适合开头/转场而非全程铺底。
- `Inferred`：可被产品安全提取的是高层风格信号（BPM、能量、器乐、旋律密度、情绪曲线），不能复制具体热门旋律、riff 或编曲结构。

## Contradictions or Gaps

- FMA FAQ 对 CC0 的“是否必须署名”表述并不严谨；本项目以 Creative Commons 官方 CC0 文本为准，同时保留自愿来源记录。
- FMA 是平台而非版权人；虽然 HoliznaCC0 持续、逐曲声明 CC0，仍无法由平台页面绝对证明完整权利链。
- 没有任何开放授权能保证永不触发平台误报；Content ID 风险只能通过证据留存、避免注册指纹和申诉流程降低。
- 当前曲目适配度尚未通过真实中文人声听测，只完成了频段、节拍、响度和人工场景标签。
- 抖音官方公开的是全品类热歌榜，没有公开“口播 BGM 榜”；本轮具体曲目来自高收藏创作者歌单，不具有统计代表性。
- 创作者把内容标为“纯音乐”时，列表仍可能包含人声歌曲或伴奏/改编版本；标题标签不能替代逐首试听与权利审计。
- 用户对首版 CC0 曲库的质量反馈与平台样本方向一致：授权安全和技术指标并不能自动带来“网感、熟悉感、专业制作感”。
