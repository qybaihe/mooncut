# MoonCut Editor

Use this skill for turning a talking-head source video into a finished MoonCut Remotion render.

1. Inspect the source before making editorial claims.
2. Use timed transcript segments as the source of truth for scene timing and on-screen copy.
3. Run the conservative generated-visual scheduler. Default to zero images, use one only when it materially explains a hard-to-source example, and never exceed two or treat generated art as evidence.
4. Generate the face track before rendering so speaker windows remain stable.
5. Save a semantic edit spec rather than writing one-off frame logic.
6. Render with the default Sonoma/native-macOS visual preset.
7. Verify the encoded MP4 and still contact sheet before declaring success.

The full production contract is in `SPEC.md` at the package root.
