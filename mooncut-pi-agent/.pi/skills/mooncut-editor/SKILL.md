# MoonCut Editor

Use this skill for turning a talking-head source video into a finished MoonCut Remotion render.

1. Inspect the source before making editorial claims.
2. Use timed transcript segments as the source of truth for scene timing and on-screen copy.
3. Generate the face track before rendering so speaker windows remain stable.
4. Save a semantic edit spec rather than writing one-off frame logic.
5. Render with the default Sonoma/native-macOS visual preset.
6. Verify the encoded MP4 and still contact sheet before declaring success.

The full production contract is in `SPEC.md` at the package root.
