MOONCUT · PI EDITING AGENT

上传口播，得到剪辑成片。

异步 API 会完成转录、真实网页/X 证据、人脸跟踪、语义分镜、Remotion 渲染和视觉质检。

1. 一次提交

curl --cacert mooncut-ca.crt \
  -H "Authorization: Bearer $MOONCUT_API_KEY" \
  -H "Content-Type: video/mp4" \
  --url-query "filename=talking-head.mp4" \
  --url-query "prompt=按默认原生macOS规范剪辑" \
  --data-binary @talking-head.mp4 \
  "https://42.194.219.172/v1/edits"

2. 查询进度

curl --cacert mooncut-ca.crt \
  -H "Authorization: Bearer $MOONCUT_API_KEY" \
  "https://42.194.219.172/v1/edit-jobs/JOB_ID"

3. 下载成片

curl --cacert mooncut-ca.crt \
  -H "Authorization: Bearer $MOONCUT_API_KEY" \
  -o final.mp4 \
  "https://42.194.219.172/v1/edit-jobs/JOB_ID/artifacts/video"
